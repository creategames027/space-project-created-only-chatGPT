# 🔭 Astronomy Research Plan

The scientific part of the project will grow as a research archive.

## Current directions

### 1. Object database
- Known astronomical objects
- New discoveries
- Interesting anomalies

### 2. Simulations
- Orbital mechanics
- Gravity
- Stars and galaxies
- Black holes

### 3. Observation tools
- Observation logs
- Object visibility
- Amateur astronomy guides

### 4. Future ideas
- Interactive star map
- Exoplanet explorer
- Space data dashboard

Goal: transform the project from a collection of files into a small digital astronomy laboratory.
